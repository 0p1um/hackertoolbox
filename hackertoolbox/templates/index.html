Welcome in hackertoolbox, this is a program for lazy hackers who want to easily run and schedule boring tasks. Select a tool bellow:

SEARCH LOCAL DATASETS
Search stuff in local datasets

This tool is able to perform very fast search in local datasets (using sift, a faster grep). You can add datasets by putting folders of your dataset in the datasets folder (/usr/share/datasets/). Dont forget to add a info.json file in your dataset folder, this file is used to gather metadata about the dataset.

OSINT Tool
